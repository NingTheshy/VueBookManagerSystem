# 《Web 应用开发》课程设计 —— 题目七：图书管理系统前端设计与开发

## 适用信息

适用年级：2025 级

适用专业：软件工程（专升本）

考核方式：课程设计

## 一、项目开发要求

本项目要求学生运用所学的 Vue3 知识完成一个图书管理系统的前端页面设计与开发；项目要求有基本的分层、结构设计合理；符合面向对象设计思想。

## 二、系统功能要求

### 1. 图书管理

包括图书的添加、编辑、删除等操作，可以记录图书的基本信息，如书名、作者、出版社、ISBN 等。

### 2. 借阅历史

提供借阅和归还图书的功能，包括借阅期限的设置、借阅历史的记录等。

### 3. 采购管理

图书馆可以记录和管理图书馆中图书的需求，包括预算、数量、类型等。图书馆可以通过系统生成采购订单，包括图书的详细信息、数量、价格等，并进行订单的跟踪和管理。

### 4. 用户信息

提供图书的査询和检索功能，读者可以根据关键词、作者、分类等条件进行搜索。

### 5. 系统管理

包括系统设置、权限管理、日志记录等功能，确保系统的安全和稳定运行。

## 三、页面参考说明

### （一）前端页面参考

1. 登录页面：包含账号输入框、密码输入框、验证码输入框、登录按钮、注册跳转文字。

2. 图书查询页面：

   页面设有图书号、图书类型、图书名称、作者名称、出版社、总数量展示列，每条图书信息配置借阅按钮；侧边栏包含热门推荐、最佳读者、问题反馈模块。

3. 当前借阅信息页面：

   展示图书号、图书名称、读者账号、读者姓名、借阅日期、截止还书日期，单条记录配置还书操作按钮，侧边栏沿用热门推荐、最佳读者、问题反馈模块。

### （二）后端管理页面参考

1. 后台登录页：后台管理系统登录界面，包含 admin 账号登录区、记住密码选项。
2. 用户信息管理页：可按用户名、电话、地址条件检索，支持新增用户，表格展示编号、用户名、电话、年龄、地址、注册时间、借阅次数、在借欠款，附带编辑、删除操作。
3. 借阅历史管理页：可按图书名称、分类、作者检索，表格展示编号、借阅人、图书名称、分类、作者、价格、是否归还字段，支持编辑操作。
4. 图书信息管理页：可通过图书名称、分类、作者、价格筛选，实现图书新增；列表展示创建时间、图书名称、分类、作者、价格、借阅数量、在馆数量，配套编辑、删除功能。
5. 采购管理页面：可填写书籍名称、采购金额进行搜索，支持新增采购单；表格展示编号、书籍名称、采购金额、采购负责人，附带编辑、删除操作。

## 四、课程设计关联考核知识点

4、Vue 内置指令、自定义指令原理

5、v-bind 样式的绑定语法、对象语法及数组语法

6、v-model 表单双向绑定、指令缩写

7、Vue 监听器、对象监听、方法监听

8、Vue 事件处理机制、处理方法、修饰符

9、Vue 组件、数据传递、组合 API

10、DOM、自定义指令

11、router 路由、命名视图

12、通信组件、状态管理

13、Element-Plus 组件

14、移动端页面开发，单页面开发技术

15、项目文档编写
